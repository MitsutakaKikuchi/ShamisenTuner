ご提示いただいた画像のような、非常に美しくリッチな質感（リアルな木目、金属的な光沢、立体感、グラデーション）を持つUIをReact Nativeの標準機能（単色塗りつぶしや単純な影）だけで完全に再現することには限界がありました。ご期待に沿えず申し訳ありません。

画像のような「和風かつ高級感のあるUI」を忠実に再現するためには、**①パーツ画像の切り出し（アセット化）** と、**②グラデーション描画ライブラリの導入** の2点が必要不可欠です。

最大限リソースを使用して画像を忠実に再現するための具体的なアプローチと、必要な素材についてお伝えします。

---

### 1. ご用意いただきたい画像素材（アセット）

CSS（プログラム）だけでは表現が難しい「質感」の部分を、透過PNG画像などでご用意いただけると、一気に完成形に近づきます。

以下のパーツの切り出しをお願いできますでしょうか？
（※すべて透過PNG形式、解像度は@2xまたは@3x程度の高画質が理想です）

1. **背景画像 (`bg_texture.jpg` または `png`)**
* 木目のテクスチャと、上部の和風パターン（青海波のような柄）が合わさった背景全体の一枚絵。


2. **ドラムロールの左右の木製キャップ (`picker_cap_left.png`, `picker_cap_right.png`)**
* 中央のピッカーの左右にある、丸みを帯びた木と金色の装飾が施されたカバー部分。


3. **微調整スライダーのツマミ (`slider_thumb.png`)**
* 中央に配置されている、ヘアライン加工のような光沢がある金色の丸いツマミ。


4. **糸ボタンの金色のリング枠 (`button_ring.png` / `button_ring_active.png`)**
* 一の糸〜三の糸のボタンの周囲にある、立体的で輝きのある金色の枠線画像。（可能であれば、通常時と光っている発音時の2パターン）

### 2. 追加したいライブラリ（グラデーション用）

画像に見られる「本調子・二上り・三下り」のタブの金色の変化や、ドラムロールの上下が丸みを帯びて暗くなる表現には、グラデーションが必須です。
以下のコマンドでExpo標準のグラデーションライブラリを追加させてください。

```bash
npx expo install expo-linear-gradient

```

#### ② `src/components/TuningModeSelector.tsx` の修正

角が四角く、選択部分だけがゴールドに輝く画像の見た目に合わせます。

```tsx
// src/components/TuningModeSelector.tsx
import React from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';
import { COLORS, FONT_SIZES, SPACING, BORDER_RADIUS } from '../constants/theme';
import type { TuningMode } from '../constants/tuningData';

type TuningModeSelectorProps = {
  modes: TuningMode[];
  selectedModeId: string;
  onModeChange: (modeId: string) => void;
};

export const TuningModeSelector: React.FC<TuningModeSelectorProps> = ({
  modes,
  selectedModeId,
  onModeChange,
}) => {
  return (
    <View style={styles.container}>
      <View style={styles.tabGroup}>
        {modes.map((mode, index) => {
          const isSelected = mode.id === selectedModeId;
          return (
            <TouchableOpacity
              key={mode.id}
              style={[
                styles.tab,
                isSelected && styles.tabActive,
                index !== modes.length - 1 && styles.borderRight, // 最後の要素以外に右線を引く
              ]}
              onPress={() => onModeChange(mode.id)}
              activeOpacity={0.8}
            >
              <Text style={[styles.tabText, isSelected && styles.tabTextActive]}>
                {mode.label}
              </Text>
            </TouchableOpacity>
          );
        })}
      </View>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    paddingHorizontal: SPACING.xl,
    paddingVertical: SPACING.md,
  },
  tabGroup: {
    flexDirection: 'row',
    borderWidth: 1.5,
    borderColor: COLORS.gold,
    borderRadius: BORDER_RADIUS.sm,
    overflow: 'hidden',
    backgroundColor: '#2b1b11', // 非選択時の暗い背景
  },
  tab: {
    flex: 1,
    paddingVertical: SPACING.lg,
    alignItems: 'center',
    justifyContent: 'center',
  },
  borderRight: {
    borderRightWidth: 1.5,
    borderRightColor: COLORS.goldDark,
  },
  tabActive: {
    backgroundColor: COLORS.surfaceHighlight, // 画像のようなゴールド
  },
  tabText: {
    fontSize: FONT_SIZES.xl,
    color: COLORS.textPrimary,
    // 明朝体のような雰囲気を出すため
    fontFamily: 'serif', 
  },
  tabTextActive: {
    color: COLORS.textDark,
    fontWeight: 'bold',
  },
});

```

#### ③ `src/components/StringPlayButtons.tsx` の修正

ボタンをより大きくし、二重の枠線で「金色のリング」を擬似的に表現します。

```tsx
// src/components/StringPlayButtons.tsx
import React, { useMemo } from 'react';
import { View, Text, TouchableOpacity, StyleSheet } from 'react-native';
import { COLORS, FONT_SIZES, SPACING } from '../constants/theme';
import { SHAMISEN_STRINGS } from '../constants/tuningData';
import { getStringNoteName } from '../utils/noteNameHelper';

type StringPlayButtonsProps = {
  activeStringId: string | null;
  baseNoteId: string;
  tuningModeId: string;
  isAutoPlaying: boolean;
  onStringToggle: (stringId: string) => void;
};

export const StringPlayButtons: React.FC<StringPlayButtonsProps> = ({
  activeStringId,
  baseNoteId,
  tuningModeId,
  isAutoPlaying,
  onStringToggle,
}) => {
  const stringNotes = useMemo(() => {
    return SHAMISEN_STRINGS.map((string) => ({
      ...string,
      noteName: getStringNoteName(string.id, baseNoteId, tuningModeId),
    }));
  }, [baseNoteId, tuningModeId]);

  return (
    <View style={styles.container}>
      {stringNotes.map((string) => {
        const isActive = activeStringId === string.id;
        return (
          <TouchableOpacity
            key={string.id}
            style={[
              styles.outerRing,
              isActive && styles.outerRingActive,
              isAutoPlaying && !isActive && styles.buttonDisabled,
            ]}
            onPress={() => onStringToggle(string.id)}
            activeOpacity={0.8}
            disabled={isAutoPlaying}
          >
            <View style={[styles.innerButton, isActive && styles.innerButtonActive]}>
              <Text style={[styles.label, isActive && styles.labelActive]}>
                {string.label}
              </Text>
              <Text style={[styles.noteName, isActive && styles.noteNameActive]}>
                {string.noteName}
              </Text>
            </View>
          </TouchableOpacity>
        );
      })}
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flexDirection: 'row',
    justifyContent: 'space-between',
    paddingHorizontal: SPACING.lg,
    paddingTop: SPACING.md,
  },
  // 画像の「太い金の縁」を模した外枠
  outerRing: {
    width: 100,
    height: 100,
    borderRadius: 50,
    backgroundColor: '#3a2311',
    borderWidth: 4,
    borderColor: '#c6a265', // ゴールド枠
    alignItems: 'center',
    justifyContent: 'center',
    shadowColor: '#000',
    shadowOffset: { width: 0, height: 6 },
    shadowOpacity: 0.8,
    shadowRadius: 8,
    elevation: 10,
  },
  outerRingActive: {
    borderColor: COLORS.activeGlow,
    shadowColor: COLORS.activeGlow,
    shadowOpacity: 1,
    shadowRadius: 15,
  },
  buttonDisabled: {
    opacity: 0.5,
  },
  // ボタンの内側の暗い部分
  innerButton: {
    width: 86,
    height: 86,
    borderRadius: 43,
    backgroundColor: '#1b0e06', // 真ん中の暗い部分
    borderWidth: 1,
    borderColor: '#5a3d24',
    alignItems: 'center',
    justifyContent: 'center',
  },
  innerButtonActive: {
    backgroundColor: '#2e1c0c',
  },
  label: {
    fontSize: FONT_SIZES.md,
    color: '#d8bd8a',
    fontFamily: 'serif',
  },
  labelActive: {
    color: '#ffdf91',
  },
  noteName: {
    fontSize: FONT_SIZES.hero,
    color: '#c6a265',
    fontFamily: 'serif',
  },
  noteNameActive: {
    color: '#ffdf91',
    textShadowColor: 'rgba(255, 223, 145, 0.8)',
    textShadowOffset: { width: 0, height: 0 },
    textShadowRadius: 10,
  },
});

```
